#object types

- number: 10, 3.14, -25, etc.
- string: "Hello", 'World', "", etc.
- boolean: True, False
- list: [1, 2, 3], ['a', 'b'], [], etc.
- tuple: (1, 2),
- dictionary: {'name': 'John', 'age': 30}
- set: {1, 2, 3}, set()
- none: None
- function, method, lambda function,module, class are objects too!
